Personal Projects

  • ExpenseBot

    A simple and clean android app to keep track of expenses. It allows users to register new expenses, to tag them and later to search through expenses by their tags. It can show a graph with the monthly summary of expenses for the current year. The graph can be restricted to the searched tags.

    This is my first android app. I built it with the purpose of learning to develop for this platform.
